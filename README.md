# Deemo to Midi Converter
Python version of deemo-to-midi converter in [my zhihu answer](https://www.zhihu.com/question/31424763/answer/51905512). The script is written in python 3.

## Usage

 - Extract Unity TextAssets (using tools like disunity)
 
 - Copy the script in the folder with `*.json.txt` and run it, or call the script with arguments `-i` and `-d`, specifying the input folder and output folder respectively .
 
 - The resulting `.mid` file will be stored in the destination folder.
